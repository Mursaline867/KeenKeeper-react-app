# KeenKeeper — Keep Your Friendships Alive

A responsive React app inspired by the provided Figma file for tracking friendships, logging quick check-ins, reviewing interaction history, and visualizing activity analytics.

## Live project summary

KeenKeeper helps users:
- see which friendships are overdue or almost due
- open a friend profile and log a quick call, text, or video interaction
- review a full interaction timeline with filtering and search
- view a Recharts analytics page for call, text, and video totals

## Technologies used

- React
- React Router DOM
- Tailwind CSS
- Recharts
- React Hot Toast
- Lucide React
- Vite

## Key features

1. **Responsive layout** for mobile, tablet, and desktop screens.
2. **Quick Check-In actions** that instantly add timeline entries and show toast notifications.
3. **Analytics dashboard** with a live pie chart driven by logged interactions.
